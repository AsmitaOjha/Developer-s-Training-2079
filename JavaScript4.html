Var

The var keyword is used to create a new variable in the current context or scope being used. Var keyword declares both local and global variables. It depends upon the way it is declared. If the var keyword is used to declare variable outside the function or block then it is declared as global variable.

When the variable is declared inside the function or block then at that time, the variable shows behavior as local variable. This means that the variable can be accessed only inside the block or that function.
